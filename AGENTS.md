# AGENTS.md — エージェント運用ルール

このリポジトリは「判断材料ドキュメント生成プロジェクト」。
正式なワークフローは `./PROJECT.md` を参照(常に最新の正)。

ここには、どのエージェント(Codex / Claude いずれも)も毎回守る**不変ルール**だけを置く。

## 不変ルール

1. **要約しない。** 目的は要約ではなく判断材料の集積。情報量を最大化する。まとめは個別指示があるときだけ。
2. **選択肢を提示する。** 有力候補を1つに断定せず、相反する見解・代替案を並列で残す。断定箇所を見つけたら選択肢を広げる方向で追加調査する。
3. **調査は最低100件・上限なし。** 著者/媒体/立場を分散させ、偏りを記録して補正する。
4. **SubAgent は生情報をファイルに書き出す。** 親へ要約を返させない。返すのは目録(件数・出典ドメイン・著者・カバー/未カバー論点)のみ。書き出し先は `./<テーマ>/research/<観点グループ>.md`。
5. **調査並列は 4〜6 体。** 近い観点をグループにまとめて割り当てる。
6. **出力は Single HTML。** 全観点を1ファイルに集約し、画像は Base64 埋め込み。目次・アンカーで飛ばし読みできるようにする。
