# Documenter — 判断材料ドキュメント生成プロジェクト

特定テーマについて、要約ではなく「判断材料」となる資料を、多様な観点・著者・媒体から大量に集積し、選択肢を提示する Single HTML として生成するプロジェクト。

ワークフローの定義は [`./PROJECT.md`](./PROJECT.md) を参照。

## 構成

```
  PROJECT.md              ワークフロー定義(調査→作成→レビュー→Single HTML化)
  <テーマ名>/
    <テーマ名>.html       成果物(画像Base64埋込・1ファイル完結)
    sources.*             調査ログ(出典・著者・媒体・観点タグ)
```

## 方針(要点)

- 1テーマあたり最低100件・上限なしで調査し、著者/媒体/立場を分散させる
- 要約せず観点別に積み上げ、相反する見解を並列で残す
- レビューは削るより広げる方向(断定的に1候補のみの箇所は選択肢を追加)
- 出力は全観点を1つの巨大HTMLに集約
